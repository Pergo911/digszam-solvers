## Usages

**relacios-solver.js**

- Illeszd be a konzolba a feladat oldalán a moodle-ben.
- Húzd az éleket a táblázat megfelelő oszlopából.

**regex_to_dfa_solver_full_workflow.js**

**Regex -> DFA** vagy **Regex -> NFA** feladatok megoldásához

- Illeszd be a konzolba a feladat oldalán a moodle-ben.
- Add meg a regexet.
- Solver lefut, és a helyes válasz form submiton lesz injektálva. (ellenőrzés gomb biztos ami biztos, de szerintem simán következő oldlara lépéssel is megy)
